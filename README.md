用于存储提示词复用，更方便用Ai。
左边创建文件，右上角”新建Prompt“是给创建的文件加入提示词。
可以加入图片。在”prompt列表点击图片可以预览放大的图片“右上角查看文字可以预览所有写的提示词。
在编辑中的“prompt内容”选中文本，然后点击“标记选中文本”或者按"空格",会标记词语。
保存后再次点击刚刚的文件，在主界面的编写框中右边会显示刚刚标记的词语，会自动放到右下角的编写框中，方便写提示词。一般Ai生图更方便。
最关键的是可以将写的所有提示词文件导出，发给其它人，导入会显示完整导出的内容。
编写框可以拖动。左边的文件点击小三角可以折叠。
只是第一个版本，后面会加入接口，通过open claw读取指定文件中的所有图片，然后自动写图片包含的提示词，并且自动标记词语的功能。
